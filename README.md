# SOMEBOT (BOT LIBRARY)

## О ПРОГРАММЕ

**Цели проекта:** Предоставить простую в использовании библиотеку для написания простых Telegram-ботов.

_Проект находится на стадии разработки._

## SOFTWARE / HARDWARE
* JDK: 17.0.2
  * Spring Boot Starter
  * Lombok, JUnit
  * Gradle
* (Docker-compose) - todo.

### Рекомендуемые параметры VPS:
